## [简介](https://github.com/OS-Q/OS-Q/wiki)

Q系统的核心思想就是强化边缘设备的作用，以云端为指挥中心，所有的数据通过云端导流到指定的设备上，降低对云端的处理性能需求，通过通过丰富的边缘端设备实现丰富的功能
